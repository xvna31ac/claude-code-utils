# コア開発サブエージェント

コア開発サブエージェントは、現代的なアプリケーションをゼロから構築するための基本ツールキットです。バックエンドサービスからフロントエンドインターフェース、モバイルアプリからデスクトップアプリケーション、シンプルなAPIから複雑な分散システムまで、開発のあらゆる領域をカバーしています。

## 🎯 コア開発サブエージェントの使い方

以下の場合にこれらのサブエージェントを使用してください：
- **新しいアプリケーションの構築** — 適切なアーキテクチャでゼロから開始
- **複雑な機能の実装** — 深い技術的専門知識が必要な場合
- **スケーラブルなシステムの設計** — 成長に対応できるシステム
- **美しいUIの作成** — 優れたユーザー体験の提供
- **リアルタイム機能の開発** — インタラクティブなアプリケーション
- **レガシーシステムの近代化** — 現在のベストプラクティスへの移行
- **パフォーマンスの最適化** — スタック全体での最適化

## 📋 利用可能なサブエージェント

### [**api-designer**](api-designer.md) — REST・GraphQL APIアーキテクト
美しく、直感的でスケーラブルなAPIを設計するアーキテクト。RESTfulの原則、GraphQLスキーマ、APIバージョニング、ドキュメント作成のエキスパート。開発者が使いやすく、将来性のあるAPIを実現します。

**使用タイミング：** 新しいAPIの設計、既存エンドポイントのリファクタリング、API標準の実装、包括的なAPIドキュメントの作成。

### [**backend-developer**](backend-developer.md) — スケーラブルなAPIのサーバーサイド専門家
堅牢なサーバーアプリケーション、RESTful API、マイクロサービス構築のスペシャリスト。データベース設計、認証システム、パフォーマンス最適化に優れています。Node.js、Python、Javaなどのバックエンド技術でアプリケーションの骨格を構築します。

**使用タイミング：** APIの構築、データベース設計、認証の実装、ビジネスロジックの処理、サーバーパフォーマンスの最適化。

### [**electron-pro**](electron-pro.md) — デスクトップアプリケーション専門家
Web技術を使ったクロスプラットフォームデスクトップアプリケーション構築のスペシャリスト。Electronフレームワークでネイティブ機能を持つインストーラブルなデスクトップアプリを作成します。自動更新、システム統合、デスクトップ固有の機能に対応。

**使用タイミング：** デスクトップアプリの作成、WebアプリのデスクトップへのポーティングやシステムトレイのImplementation、オフライン対応デスクトップツールの構築。

### [**frontend-developer**](frontend-developer.md) — React・Vue・Angular向けUI/UXスペシャリスト
レスポンシブでアクセシブル、高性能なユーザー体験を作るモダンWebインターフェースのマスター。コンポーネントアーキテクチャ、状態管理、モダンCSSのエキスパート。デザインをピクセルパーフェクトなインタラクティブアプリに変換します。

**使用タイミング：** Webインターフェースの作成、複雑なUIコンポーネントの実装、フロントエンドパフォーマンスの最適化、アクセシビリティ対応。

### [**fullstack-developer**](fullstack-developer.md) — エンドツーエンドの機能開発
スタック全体をシームレスに扱う万能エキスパート。データベースからUIまでの完全な機能を構築し、フロントエンドとバックエンドの滑らかな統合を実現。迅速なプロトタイピングや完全な機能実装に最適。

**使用タイミング：** 完全な機能の構築、アプリケーションのプロトタイピング、小〜中規模プロジェクト、スタック全体での統一的な開発。

### [**graphql-architect**](graphql-architect.md) — GraphQLスキーマ・フェデレーション専門家
スキーマ設計からフェデレーション戦略まで、GraphQLエコシステムに特化。リゾルバ最適化、サブスクリプションパターン、GraphQLベストプラクティスのマスター。柔軟で効率的なデータ層の構築に最適。

**使用タイミング：** GraphQL APIの実装、スキーマ設計、リゾルバの最適化、フェデレーションのセットアップ、RESTからGraphQLへの移行。

### [**microservices-architect**](microservices-architect.md) — 分散システム設計者
マイクロサービスアーキテクチャの設計と実装のエキスパート。サービスの分解、サービス間通信、分散トランザクション、オーケストレーションを担当。システムが水平スケーリングと高い可用性で動作することを保証します。

**使用タイミング：** モノリスのマイクロサービス分割、分散システムの設計、サービスメッシュの実装、分散システムの課題解決。

### [**mobile-developer**](mobile-developer.md) — クロスプラットフォームモバイルスペシャリスト
iOS・Androidのネイティブおよびクロスプラットフォームモバイルアプリ作成のエキスパート。React Native、Flutter、ネイティブ開発に精通。オフライン機能、プッシュ通知、アプリストア最適化などモバイル固有の課題に対応。

**使用タイミング：** モバイルアプリの構築、モバイル固有機能の実装、モバイルパフォーマンスの最適化、アプリストアへの申請準備。

### [**ui-designer**](ui-designer.md) — ビジュアルデザイン・インタラクション専門家
美しく直感的でアクセシブルなユーザーインターフェースを作るビジュアルデザインのマスター。デザインシステム、タイポグラフィ、カラー理論、インタラクションパターンのエキスパート。美しさと機能性を両立させながらブランドの一貫性を維持します。

**使用タイミング：** ビジュアルデザインの作成、デザインシステムの構築、インタラクションパターンの定義、ブランドアイデンティティの確立、開発向けデザインハンドオフ。

### [**websocket-engineer**](websocket-engineer.md) — リアルタイム通信専門家
リアルタイムの双方向通信のマスター。WebSocketサーバーの実装、大規模な接続管理、チャット・通知・ライブ更新などのリアルタイム機能に対応。Socket.ioとネイティブWebSocketの両方を熟知。

**使用タイミング：** チャットアプリの構築、リアルタイム通知の実装、コラボレーション機能の作成、ライブ更新ダッシュボードの開発。

### [**wordpress-master**](wordpress-master.md) — WordPress開発・最適化専門家
シンプルなブログからエンタープライズプラットフォームまで構築できるWordPressエコシステムのスペシャリスト。テーマ開発、プラグインアーキテクチャ、Gutenbergブロック、パフォーマンス最適化のマスター。クラシックなPHP開発とモダンなブロックベースのソリューションの両方に精通。

**使用タイミング：** WordPressサイトの構築、カスタムテーマの開発、プラグインの作成、WooCommerceソリューションの実装、WordPressパフォーマンスの最適化。

## 🎬 はじめかた

1. **適切なサブエージェントを選択** — 具体的なニーズに基づいて選ぶ
2. **明確なコンテキストを提供** — プロジェクト要件を明確に説明する
3. **技術スタックを指定** — 好みがあれば明示する
4. **制約を説明** — パフォーマンス、スケーラビリティ、タイムラインなど
5. **サブエージェントに従う** — ベストプラクティスと実装をガイドしてもらう

各サブエージェントには以下が含まれます：
- ドメインの深い専門知識
- 現在のベストプラクティスの知識
- 既存のコードベースへの対応能力
- クリーンで保守性の高いコードへのこだわり
- プロダクション要件の理解

## 📚 ベストプラクティス

- **アーキテクチャから始める：** 実装前にアーキテクト（API、GraphQL、マイクロサービス）を使用する
- **頻繁にイテレーションする：** 短いサイクルでサブエージェントと作業する
- **専門知識を組み合わせる：** 複雑なプロジェクトには複数のサブエージェントを活用する
- **慣例に従う：** 各サブエージェントはドメインのベストプラクティスを知っている
- **プロダクション品質を意識：** 全サブエージェントはスケーラビリティ、セキュリティ、保守性を考慮する

サブエージェントを選んで、素晴らしいアプリケーションを今すぐ構築しましょう！
