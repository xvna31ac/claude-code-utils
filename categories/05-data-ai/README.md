# データ・AIサブエージェント

データ・AIサブエージェントは、データエンジニアリング、機械学習、人工知能の世界における専門家です。堅牢なデータパイプラインの構築から高度なMLモデルのトレーニング、データベースの最適化からスケールでのAIシステムのデプロイまで、あらゆる領域をカバーします。生データからインテリジェントなアプリケーションへの橋渡しを担い、データ駆動型ソリューションを効率的・スケーラブル・インパクトのあるものにします。

## データ・AIサブエージェントを使うタイミング

以下のような場合に使用します：
- ETL/ELTワークフロー用の**データパイプラインの構築**
- 予測とインサイトのための**機械学習モデルのトレーニング**
- 本番デプロイ向け**AIシステムの設計**
- スケールでの**データベースパフォーマンスの最適化**
- テキスト処理のための**NLPソリューションの実装**
- **コンピュータビジョン**アプリケーションの作成
- MLOpsベストプラクティスによる**MLモデルのデプロイ**
- ビジネスインサイトのための**データ分析**

## 利用可能なサブエージェント

### [**ai-engineer**](ai-engineer.md) - AIシステム設計・デプロイエキスパート
本番対応の人工知能ソリューションを構築するAIシステムスペシャリスト。モデルのデプロイ、スケーリング、統合をマスターしています。AIリサーチと現実世界のアプリケーションの橋渡しをします。

**使用タイミング：** AIモデルの本番デプロイ、AIシステムアーキテクチャの設計、AIのアプリケーション統合、AIサービスのスケーリング、AIパイプラインの実装時。

### [**data-analyst**](data-analyst.md) - データインサイト・ビジュアライゼーションスペシャリスト
データを実用的なインサイトに変換するアナリティクスエキスパート。統計分析、データビジュアライゼーション、ビジネスインテリジェンスツールをマスターしています。データで説得力のあるストーリーを語ります。

**使用タイミング：** ビジネスデータの分析、ダッシュボードの作成、統計分析の実施、レポートの構築、データインサイトの発見時。

### [**data-engineer**](data-engineer.md) - データパイプラインアーキテクト
スケーラブルなデータパイプラインを構築するデータインフラスペシャリスト。ETL/ELTプロセス、データウェアハウジング、ストリーミングアーキテクチャのエキスパート。ソースからインサイトまでデータを確実に流します。

**使用タイミング：** データパイプラインの構築、データアーキテクチャの設計、ETLプロセスの実装、データウェアハウスの設定、ビッグデータ処理時。

### [**data-scientist**](data-scientist.md) - アナリティクス・インサイトエキスパート
統計学、機械学習、ドメイン知識を組み合わせるデータサイエンス実践者。予測モデリング、実験、高度なアナリティクスをマスターしています。複雑なデータセットから価値を抽出します。

**使用タイミング：** 予測モデルの構築、実験の実施、高度なアナリティクスの実施、MLアルゴリズムの開発、複雑なデータ問題の解決時。

### [**database-optimizer**](database-optimizer.md) - データベースパフォーマンススペシャリスト
クエリを高速実行するデータベースパフォーマンスエキスパート。インデックス戦略、クエリ最適化、データベースチューニングをマスターしています。データベースをピークパフォーマンスで稼働させます。

**使用タイミング：** 遅いクエリの最適化、効率的なスキーマ設計、インデックス戦略の実装、データベースパフォーマンスのチューニング、データベースのスケーリング時。

### [**llm-architect**](llm-architect.md) - 大規模言語モデルアーキテクト
LLMソリューションを設計・デプロイするLLMスペシャリスト。プロンプトエンジニアリング、ファインチューニング、LLMアプリケーションのエキスパート。現代の言語モデルの力を活用します。

**使用タイミング：** LLMソリューションの実装、プロンプト戦略の設計、モデルのファインチューニング、チャットボットの構築、AI駆動アプリケーションの作成時。

### [**machine-learning-engineer**](machine-learning-engineer.md) - 機械学習システムエキスパート
データからデプロイまでのMLライフサイクル全体を構築するMLエンジニアリングスペシャリスト。本番環境でモデルが確実に動作するよう保証します。

**使用タイミング：** MLパイプラインの構築、MLシステムの実装、モデルのデプロイ、MLインフラの作成、MLソリューションの本番化時。

### [**ml-engineer**](ml-engineer.md) - 機械学習スペシャリスト
MLモデルを開発・最適化する機械学習エキスパート。様々なアルゴリズム、フレームワーク、テクニックに精通しています。機械学習で複雑な問題を解決します。

**使用タイミング：** MLモデルのトレーニング、アルゴリズムの選択、モデルパフォーマンスの最適化、MLソリューションの実装、新技術の実験時。

### [**mlops-engineer**](mlops-engineer.md) - MLOpsおよびモデルデプロイエキスパート
MLモデルのデプロイと運用をスムーズにするMLOpsスペシャリスト。MLのCI/CD、モデルモニタリング、バージョニングをマスターしています。DevOpsプラクティスを機械学習に適用します。

**使用タイミング：** MLパイプラインの設定、モデルモニタリングの実装、MLワークフローの自動化、モデルバージョン管理、MLOpsプラクティスの確立時。

### [**nlp-engineer**](nlp-engineer.md) - 自然言語処理エキスパート
人間の言語を理解・生成するシステムを構築するNLPスペシャリスト。テキスト処理、言語モデル、言語分析のエキスパート。機械にテキストを理解させます。

**使用タイミング：** テキスト処理システムの構築、チャットボットの実装、感情分析、テキストからの情報抽出、言語理解機能の開発時。

### [**postgres-pro**](postgres-pro.md) - PostgreSQLデータベースエキスパート
高度な機能と最適化をマスターするPostgreSQLスペシャリスト。複雑なクエリ、パフォーマンスチューニング、PostgreSQL固有の機能のエキスパート。PostgreSQLの全機能を引き出します。

**使用タイミング：** PostgreSQLでの作業、Postgresクエリの最適化、高度な機能の実装、PostgreSQLスキーマの設計、Postgresの問題解決時。

### [**prompt-engineer**](prompt-engineer.md) - プロンプト最適化スペシャリスト
AIモデル向けの効果的なプロンプトを設計するプロンプトエンジニアリングエキスパート。プロンプト設計、テスト、最適化をマスターしています。戦略的なプロンプティングでAIモデルのパフォーマンスを最大化します。

**使用タイミング：** LLM向けプロンプトの設計、AIレスポンスの最適化、プロンプト戦略の実装、プロンプト効果のテスト、プロンプトベースアプリケーションの構築時。

## データ・AI共通パターン

**エンドツーエンドMLシステム：**
- データパイプラインは **data-engineer**
- モデル開発は **data-scientist**
- モデル最適化は **ml-engineer**
- デプロイは **mlops-engineer**

**AIアプリケーション：**
- LLM統合は **llm-architect**
- プロンプト最適化は **prompt-engineer**
- システム設計は **ai-engineer**
- テキスト処理は **nlp-engineer**

**データプラットフォーム：**
- インフラは **data-engineer**
- パフォーマンスは **database-optimizer**
- PostgreSQLは **postgres-pro**
- インサイトは **data-analyst**

**本番ML：**
- MLシステムは **machine-learning-engineer**
- 運用は **mlops-engineer**
- デプロイは **ai-engineer**
- データフローは **data-engineer**

## はじめ方

1. データ・AIの**目標を明確に定義する**
2. データランドスケープと要件を**評価する**
3. ニーズに合った**適切なスペシャリストを選ぶ**
4. データのコンテキストと制約を**提供する**
5. 実装の**ベストプラクティスに従う**

## ベストプラクティス

- **データ品質から始める：** 良いモデルには良いデータが必要
- **素早くイテレートする：** MLは本質的に実験的
- **すべてをモニタリングする：** モデルはドリフトし、データは変化する
- **バージョン管理する：** データ、コード、モデルを追跡する
- **徹底的にドキュメント化する：** MLシステムは複雑
- **厳密にテストする：** 本番前にモデルを検証する
- **段階的にスケールする：** 小さく始め、価値を証明する
- **倫理を守る：** AIの影響を考慮する
