# 専門ドメイン サブエージェント

専門ドメインサブエージェントは、特定の技術分野や業界の専門家です。これらのスペシャリストは、ドメイン固有の課題、規制、ベストプラクティスに関する深い知識を持っています。ブロックチェーンやIoTからフィンテックやゲーミングまで、専門分野の固有の要件とパターンを理解し、これらの複雑なドメインで優れたアプリケーションの構築を支援します。

## 専門ドメインサブエージェントを使用するタイミング

以下の場合にこれらのサブエージェントを使用します：
- **ブロックチェーンアプリケーション**とスマートコントラクトを構築する
- **IoTソリューション**を接続デバイス向けに開発する
- **決済システム**を各種プロバイダーで作成する
- **ゲームアプリケーション**をリアルタイム機能で構築する
- **フィンテックソリューション**をコンプライアンス対応で実装する
- **組み込みシステム**をハードウェア制約の中で開発する
- **モバイルアプリケーション**をネイティブ機能付きで作成する
- **金融アルゴリズム**を取引システム向けに設計する

## 利用可能なサブエージェント

### [**api-documenter**](api-documenter.md) - APIドキュメントスペシャリスト
開発者向けのAPIドキュメントを作成するAPIドキュメントの専門家。OpenAPI/Swagger、インタラクティブドキュメント、APIベストプラクティスに精通。APIを発見しやすく統合しやすくします。

**使用する場面：** REST APIのドキュメント化、API仕様の作成、開発者ポータルの構築、クライアントSDKの生成、またはAPIの発見性の向上。

### [**blockchain-developer**](blockchain-developer.md) - Web3・暗号通貨スペシャリスト
分散型アプリケーションとスマートコントラクトを構築するブロックチェーンの専門家。Ethereum、Solidity、Web3技術に精通。セキュアで効率的なブロックチェーンソリューションを作成します。

**使用する場面：** dAppの構築、スマートコントラクトの作成、DeFiプロトコルの実装、NFTプラットフォームの作成、またはブロックチェーン機能の統合。

### [**embedded-systems**](embedded-systems.md) - 組み込み・リアルタイムシステムの専門家
制約のある環境で動作する組み込みシステムのスペシャリスト。マイクロコントローラー、RTOS、ハードウェアインターフェースの専門家。リソース制限のあるデバイス向けの効率的なソフトウェアを構築します。

**使用する場面：** マイクロコントローラーのプログラミング、ファームウェアの開発、リアルタイムシステムの実装、メモリ/電力の最適化、またはハードウェアとのインターフェース構築。

### [**fintech-engineer**](fintech-engineer.md) - 金融テクノロジースペシャリスト
セキュアでコンプライアンス対応の金融アプリケーションを構築するフィンテックの専門家。決済処理、規制要件、金融APIに精通。複雑な金融テクノロジーの世界をナビゲートします。

**使用する場面：** 決済システムの構築、銀行機能の実装、金融コンプライアンスの確保、金融APIの統合、または取引プラットフォームの開発。

### [**game-developer**](game-developer.md) - ゲーム開発の専門家
魅力的なインタラクティブエクスペリエンスを作成するゲーミングスペシャリスト。ゲームエンジン、リアルタイムネットワーキング、パフォーマンス最適化の専門家。プレイヤーを魅了するゲームを構築します。

**使用する場面：** ゲームの開発、ゲームメカニクスの実装、ゲームパフォーマンスの最適化、マルチプレイヤー機能の構築、またはゲームツールの作成。

### [**iot-engineer**](iot-engineer.md) - IoTシステム開発者
物理デバイスをクラウドに接続するIoTの専門家。デバイスプロトコル、エッジコンピューティング、IoTプラットフォームに精通。モノのインターネットのためのスケーラブルなソリューションを作成します。

**使用する場面：** IoTアプリケーションの構築、デバイス通信の実装、IoTフリートの管理、センサーデータの処理、またはIoTアーキテクチャの設計。

### [**mobile-app-developer**](mobile-app-developer.md) - モバイルアプリケーションスペシャリスト
ネイティブおよびクロスプラットフォームアプリケーションを作成するモバイルの専門家。iOS/Android開発、モバイルUI/UX、アプリストアデプロイに精通。ユーザーがデバイスで愛用するアプリを構築します。

**使用する場面：** モバイルアプリの作成、ネイティブ機能の実装、モバイルパフォーマンスの最適化、オフライン機能の処理、またはアプリストアへのデプロイ。

### [**payment-integration**](payment-integration.md) - 決済システムの専門家
各種決済プロバイダーと方法を統合する決済スペシャリスト。PCI準拠、決済セキュリティ、トランザクション処理の専門家。決済をシームレスかつセキュアにします。

**使用する場面：** 決済ゲートウェイの統合、サブスクリプションの実装、PCI準拠の処理、トランザクションの処理、またはチェックアウトフローの構築。

### [**quant-analyst**](quant-analyst.md) - 定量分析スペシャリスト
金融アルゴリズムとモデルを開発する定量の専門家。統計分析、リスクモデリング、アルゴリズム取引に精通。市場データを利益をもたらす戦略に変換します。

**使用する場面：** 取引アルゴリズムの構築、リスクモデルの開発、金融データの分析、定量戦略の実装、またはバックテストシステムの開発。

### [**risk-manager**](risk-manager.md) - リスク評価・管理の専門家
さまざまなリスクを特定し軽減するリスク管理スペシャリスト。リスクモデリング、コンプライアンス、軽減戦略の専門家。システムとビジネスを潜在的な脅威から保護します。

**使用する場面：** 技術的リスクの評価、リスクコントロールの実装、リスクモデルの構築、コンプライアンスの確保、またはリスク管理システムの開発。

### [**seo-specialist**](seo-specialist.md) - 検索エンジン最適化の専門家
検索最適化を通じてオーガニックトラフィックを促進するSEOの専門家。テクニカルSEO、コンテンツ戦略、リンク構築に精通。データ駆動の戦略で検索ランキングと可視性を向上させます。

**使用する場面：** 検索エンジンの最適化、構造化データの実装、サイト速度の改善、コンテンツ戦略の構築、または検索パフォーマンスの分析。

## 共通ドメインパターン

**フィンテックアプリケーション：**
- コンプライアンスには **fintech-engineer**
- 決済には **payment-integration**
- リスク評価には **risk-manager**
- アルゴリズムには **quant-analyst**

**IoTプラットフォーム：**
- アーキテクチャには **iot-engineer**
- デバイスには **embedded-systems**
- アプリには **mobile-app-developer**
- APIには **api-documenter**

**ブロックチェーンプロジェクト：**
- スマートコントラクトには **blockchain-developer**
- 金融機能には **fintech-engineer**
- セキュリティには **risk-manager**
- 統合には **api-documenter**

**ゲーミングプラットフォーム：**
- ゲームロジックには **game-developer**
- モバイルには **mobile-app-developer**
- マネタイズには **payment-integration**
- ゲームAPIには **api-documenter**

**Eコマースプラットフォーム：**
- オーガニックトラフィックには **seo-specialist**
- チェックアウトには **payment-integration**
- モバイルコマースには **mobile-app-developer**
- 不正防止には **risk-manager**

## はじめに

1. **ドメイン要件と制約を理解する**
2. **適切なスペシャリストを選択する**
3. **該当する場合は規制コンプライアンスを考慮する**
4. **ドメイン固有の課題を早期に計画する**
5. **開発全体を通じてドメインの専門知識を活用する**

## ベストプラクティス

- **ドメイン知識が重要：** 分野を深く理解する
- **コンプライアンスが重要：** 多くのドメインに規制がある
- **セキュリティファースト：** 専門ドメインはしばしば機密データを扱う
- **パフォーマンス要件：** 各ドメインに固有のニーズがある
- **ユーザーの期待：** ドメインユーザーには特定のワークフローがある
- **業界標準：** 確立されたパターンに従う
- **最新情報を維持：** 専門ドメインは急速に進化する
- **徹底的にテスト：** ドメイン固有のエッジケースが重要

専門ドメインの専門家を選んで、業界をリードするアプリケーションを構築しましょう！
